# obstacle-avoider-and-ir-avoider
Arduino project to build a robot that can avoid obstacles while driving and avoid black lines


## Materials
-1 Arduino
-H Bridge L298N
-Basic car kit
-Arduino Car chasis kit
-1 9V Battery
-1 Mini-protoboard
-1 sg-90 servo motor
-1 HC-SR04 ultrasonic sensor
-1 or 2 TCRT-500 IR sensor
